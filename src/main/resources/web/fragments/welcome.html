Introduction

Oracle Coherence is an in-memory data grid enables application developers and managers fast access to key-value data. Coherence ensures for customers maximum scalability and performance in enterprise applications by providing clustered low-latency data storage, polyglot grid computing, and asynchronous event streaming.

Demonstration

The demonstrations showcases Coherence general features, scalability capabilities including:
  • Clustering and Data Sharding
  • Scalability and High Availability
  • Disk-Based Persistence
  • Parallel Queries
  • Efficient Aggregation
  • In-Place Processing
  • Federation (Grid Edition feature only)
  • Lambda Support
  • OpenTracing Support
  • Polyglot client access from JavaScript, Python and Go
  • Listening for events using Server Sent Events (SSE)
The UI is built using AngularJS 1.7.5, Bootstrap 3.3.4, as well as a number of other frameworks, and interacts with Coherence via the Coherence REST API.

Demo Insight

When Demo Insight is enabled, helpful information will be displayed in a dialog when operations are performed, or you click on the "i" icon, to help to understand what is happening under the covers as the application runs. You can disable this option from the tools menu.
