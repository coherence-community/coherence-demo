Introduction

Oracle Coherence is the industry leading in-memory data grid solution that enables organizations to predictably scale mission-critical applications by providing fast access to frequently used data. As data volumes and customer expectations increase, driven by the “internet of things”, social, mobile, cloud and always-connected devices, so does the need to handle more data in real-time, offload over-burdened shared data services and provide availability guarantees.

Demonstration

This demonstration showcases Coherence reliability, scalability, availability and performance capabilities as well as:
  • Parallel grid-wide query processing and aggregation
  • Cache Persistence - creating and recovering snapshots
  • Federation - replicating data between separate clusters
  • Java 8 Support
  • Coherence-REST support
  • Oracle Bedrock - for starting of cache servers
The UI is built using AngularJS 1.7.5, Bootstrap 3.4.0, as well as a number of other frameworks, and interacts with Coherence via the Coherence REST API.
